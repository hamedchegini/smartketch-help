This user manual is designed to help new users navigate Smartketch, a web-based platform for molecular sketching, 3D visualization, AI-assisted design, and chemical property analysis. Each section includes clear steps and screenshots.

Smartketch opens by default in the 2D editor, which is the core area for molecular sketching. Users can draw atoms, rings, and bonds using the left toolbar. The editor supports organic and inorganic structures with automatic valence checking.

Accessing Drawing Tools

  • The left toolbar contains tools for atoms, bonds, rings, and selection.
  • Click to activate a tool; the cursor will apply it to the canvas.

Adding Atoms and Bonds

  • Select an element and click to place an atom.
  • Click and drag to create a bond and a new atom.
  • Cycle bond types by clicking multiple times on a bond.
